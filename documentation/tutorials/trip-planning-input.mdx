### 1. Input Schema

First, we define what inputs our task expects:

```yaml
input_schema:
  type: object
  properties:
    locations:
      type: array
      items:
        type: string
```

This schema specifies that our task expects an array of location strings (e.g., `["New York", "London", "Paris"]`).
